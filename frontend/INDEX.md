## 🆘 Need Help?

### Quick Fixes
- **Can't install?** → Check Node.js version (18+)
- **Can't login?** → Use demo credentials (demo/demo1234)
- **Backend errors?** → Verify backend is running at port 9000
- **Build errors?** → Delete `node_modules`, reinstall
